📊 **Project: Loan Approval Prediction using Machine Learning**

In this project, I developed a machine learning model to predict loan approval decisions based on borrower characteristics. The project involved data preprocessing, handling class imbalance, model training, evaluation, and business-oriented interpretation of results.

🔹 Key Highlights:
✅ Data cleaning and preprocessing
✅ Missing value treatment and feature encoding
✅ Feature scaling using StandardScaler
✅ Class imbalance handling using SMOTE
✅ Model comparison using:
• Logistic Regression
• Random Forest
• XGBoost

✅ Performance evaluation using:
• Precision
• Recall
• F1-Score
• ROC-AUC

📈 Key Learnings:

* Building an end-to-end machine learning pipeline
* Understanding the impact of imbalanced data on model performance
* Comparing multiple classification algorithms
* Extracting actionable business insights from predictive models
